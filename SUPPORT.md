# Support

## Getting Help

Use GitHub Issues for bugs and feature requests:

- Bug report: `.github/ISSUE_TEMPLATE/bug_report.md`
- Feature request: `.github/ISSUE_TEMPLATE/feature_request.md`

For security concerns, follow `SECURITY.md` and report privately.

## What to Include

To get help quickly, please include:

- Exact steps to reproduce
- Browser/OS and version
- Relevant logs or screenshots
- Whether you are running Docker or local development setup

## Project Status

This is an actively maintained personal/open-source project. Best-effort
support is provided by maintainers and contributors.
